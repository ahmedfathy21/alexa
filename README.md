# Voice Assistant Alexa

## Features

- Voice recognition and response
- Music playback
- Smart home control
- Weather updates
- News briefings
- Timers and alarms
- Shopping lists
- Integration with third-party services
- Personalized recommendations
- Voice calling and messaging
- Skills and app integrations
- Regular updates and improvements

### The Libraries used 
- gtts
- vlc
- speech_recognition
- pyttsx3
- datetime
- os
- webbrowser
- time
- pyautogui
- pynput
- requests
- bs4
- pywhatkit
- wikipedia
